Este ficheiro descreve a estrutura de pastas presentes na pasta da solução EDM

Pasta "3D" - Contém um exemplo de um dicionário de dados (3d.xml) bem como o schema que o valida (3dValidator.xsd).

Pasta "EDM.FoudationClasses" - Contém o projecto "FoudationClasses". A framework de suporte à execução do código gerado pela solução.

Pasta "EDM.Generator" - Contém o projecto "Generator". O gerador de código.

Pasta "EDM.GeneratorEntryPoint" - Contém o projecto do "Entry Point" da solução.

Pasta "EDM.Template" - Contém o projecto "Template". 

Pasta "EDM.TemplateInstaller" - Contémo projecto "Template Installer".

Pasta "EDM.TemplateSetup" - Contém o projecto "TemplateSetup".

Pasta "EDM.Tester" - Contém os testes unitários realizados sobre o código fonte produzido.
